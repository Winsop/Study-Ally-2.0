# Study Ally Pro 2.0

Study Ally Pro is an AI-powered academic productivity platform designed to help students organize subjects, track progress, plan revision schedules, manage deadlines, and maintain consistent study habits.

## Live Demo

https://study-ally-2-0.vercel.app/

## Features

* Subject and exam management
* Progress tracking dashboard
* Smart study planning
* Spaced revision system
* Academic analytics
* Focus and productivity tools
* Study streak tracking
* Responsive modern UI

## Tech Stack

* React
* TypeScript
* Vite
* Node.js
* Vercel

## Project Goal

Study Ally was created to provide students with a centralized workspace for planning, tracking, and improving their academic performance while maintaining consistent study habits.
